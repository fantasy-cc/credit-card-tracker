# CouponCycle 版本更新记录 / Version Update Notes

**当前版本 / Current version: v1.16**

---

## 格式说明 / Format

每条更新为一行，格式：`V版本号: 简短中文描述（可多句，用逗号或句号分隔）`

---

## 历史记录 / History

V1.16：福利页「按卡 ROI」改为可折叠区块（默认收起），不挡视线。修复 ROI 计算：以往标记为完成但未记录金额的福利现按全额计入已使用。移动端折叠支持触控、加大点击区域并带展开/收起无障碍说明。

V1.15：仪表盘新增「7 天内到期」专区，快过期福利一目了然；仪表盘可直接标记完成或部分完成，不用跳转福利页。福利页支持搜索和按类别/卡片筛选。福利卡片可添加备注，方便记录使用情况。

V1.14：支持福利部分完成追踪，并精简视图选项。福利可记录部分使用金额（如 $30/$50）、显示进度条，ROI 按实际使用金额计算；新增「Mark Complete」与「Partial Complete」按钮。福利展示仅保留「按信用卡分组」与「按类别分组」，移除列表视图。

V1.13: 自定义福利提醒

V1.12: 优化卡片添加/编辑页面：可折叠福利展示、智能提示工具、信用卡昵称功能，界面更简洁易用

V1.11: 修复了些bug并且支持了新的Plati，老用户自动更新为新Plat的tracker

V1.10: 正式上线Amex 5位尾号支持，现有的已经填了4位尾号的还是会正常显示，但是Edit成5位尾号。

V1.9：增加了新CSR（旧CSR自动替换为新CSR），加入了新的Atmos御三家

V1.8：加入了Citi Strata Elite，同时加入了一个Suggest功能， 可以提供修改建议（信用卡信息，以及福利信息），然后由Admin Approve或者Reject。有兴趣做Admin的坛友可以私信我。

V1.7: 加入了一个隐藏的小功能，在Notification界面点击可以进入Loyalty Programs登记后会在点数到期前发邮件提醒，欢迎测试反馈

V1.6: 修复了添加卡片月份会往前显示一个月的问题.

V1.5: 将按钮替换为三个独立的视图按钮（列表视图、按类别分组、按信用卡分组），并移除了拖拽重排功能，使界面更清晰直观。

V1.4: 一个对于提高懒人效率的尝试: 加入了对Benefits的分组, 通过单击高效地完成逻辑分组内的所有福利 (比如一次性把所有Uber, Dinning都完成).

V1.3: 更新了Notification设置成功的提示，增加了很多卡，让卡片页面更好看了一点

V1.2: 更新Gold年费，社区维护指南，以及增强搜索页面支持模糊搜索

V1.1 加入了UA，Delta，Southwest全系列的支持

V0.5: Fix了有同名卡年费计算的问题，同时在欢迎界面加入了支持卡的列表

V0.3: 加入了年费ROI计算，更新了一下UI，以及重新排序Benefits的功能

V0.1: 今天新增了一个小功能，可以导出信用卡列表，也可以导入，更方便管理了
